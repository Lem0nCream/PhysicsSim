# Physics Electromagnetism Simulation
A Sandbox where usres can place objects with electric & magnetic properties to observe the emergent properties which arise.
Contains support for electric forces, magnetic forces, lorentz forces, collision detection & conservation of momentum, and simple-moderate circuits.

## Requirements
- .NET 10 SDK
## Running the Simulation
1. Clone the repository
```
git clone https://github.com/Lem0nCream/PhysicsSim.git
cd PhysicsSimApp
```
2. Run the simulation
```
dotnet restore
dotnet watch run
```
